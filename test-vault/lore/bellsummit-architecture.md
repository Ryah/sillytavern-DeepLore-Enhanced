# Bellsummit Architecture

Bellsummit's buildings are notable for their integration of arcane warding into structural design — the Oracles tower's outer walls incorporate resonance-dampening stone quarried from a site that no longer exists. The Triumvirate administrative complex uses a pre-Calamity foundation that was never fully understood; the basement level has been sealed since Year 203 following an incident the official record describes only as "structural instability."
