# True Names — Theory and Danger

The pre-Calamity civilization believed that every sufficiently complex entity possesses a true name — a fundamental identifier that grants significant power over the entity to any speaker. True names for humans are theoretically derivable but practically inaccessible. True names for certain non-human entities are different: they are fixed, discoverable, and extraordinarily dangerous.

The [[Oracles of Might]] classify all true name research as restricted material.
