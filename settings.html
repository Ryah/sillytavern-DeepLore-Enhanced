Slash Commands

/dle-help — Show all commands
/dle-refresh — Force re-index vault
/dle-status — Show connection & index status
/dle-browse — Browse all indexed entries
/dle-context — Preview what would be injected now
/dle-health — Audit entries for issues (30+ checks)
/dle-inspect — Show last pipeline trace
/dle-analytics — Entry usage statistics
/dle-simulate — Replay chat showing entry activation
/dle-graph — Visualize entry relationships
/dle-review [question] — Send vault to AI for review
/dle-newlore — AI suggests new lorebook entries
/dle-optimize-keys [name] — Optimize entry keywords
/dle-summarize — Generate summaries for entries
/dle-import — Import ST World Info to Obsidian
/dle-setup — Run setup wizard
/dle-notebook — Open the Author's Notebook editor
/dle-scribe [focus] — Write session note to Obsidian
/dle-scribe-history — View all session notes
/dle-pin <title> — Pin entry for this chat
/dle-block <title> — Block entry for this chat
/dle-pins — List active pins & blocks
/dle-set-era <era> — Set current era
/dle-set-location <loc> — Set current location
/dle-set-scene <type> — Set scene type
/dle-set-characters <names> — Set present characters
/dle-context-state — Show current context state
