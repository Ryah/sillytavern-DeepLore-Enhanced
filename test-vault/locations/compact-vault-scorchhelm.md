# Compact Vault

Three levels below the [[Scorchhelm]] fortress, accessible only with three Triumvirate keys plus Academy authorization. Stores the original [[Binding Compact]] document and several other foundational treaties. The [[Solarguard]] guards the entrance but is not permitted to enter.
