# Mage Dueling Code

The Mage Dueling Code governs formal arcane disputes between licensed practitioners. A sanctioned duel requires: a licensed arbiter, a registered venue with containment wards, mutual agreement on power tier limits, and a signed waiver from both parties. Unsanctioned dueling within city limits carries a two-year suspension. The Code has been amended seventeen times; the current version bans seven specific sigil combinations that previous versions permitted.

Scholars connect this to the [[Nightwatch Protocol]].
