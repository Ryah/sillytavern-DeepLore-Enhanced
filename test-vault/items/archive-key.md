# Archive Master Key

A single physical key that opens every lock in the Archive's restricted collection. There are, officially, two copies. [[Gorduin Wynlar]] holds one. The other is in the Head Archivist Kettleback's personal safe. A third copy is rumored to exist. The Archive's security protocol assumes this rumor is true.

A [[Speaking Stone]] pair is kept in the same vault, linked to an unknown counterpart outside the Archive.
