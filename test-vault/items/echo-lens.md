# Echo Lens

A monocle-sized lens that, when pressed against a surface, replays the last significant visual event experienced by that surface — specifically, the last thing that moved past it rapidly. Useful for crime scene reconstruction. Limited to approximately a 48-hour lookback window. Evidence obtained via Echo Lens is inadmissible in Triumvirate courts.

The only known working example is held by a private collector in [[Bellsummit]].
